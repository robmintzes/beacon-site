Mac installation note

This app is not signed with an official Apple Developer certificate. macOS may block it from opening. To fix this, open Terminal and run:

xattr -cr /path/to/BeaconMac.app

Then open the app normally.

🔒

Privacy

Beacon requires Accessibility access to track cursor position and detect key presses for the on-screen HUD. It does not record your screen, log keystrokes, or transmit any data — mouse coordinates and key names are used only to drive the visual overlay and are never written to disk or sent anywhere.

⊞

Windows — System Tray

Beacon for Windows runs from the System Tray. After launching, look for the Beacon icon in the tray area at the right side of the taskbar. If it isn't visible, click the arrow to expand hidden tray icons.

⌘

Mac — menu bar icon placement

On first launch, the Beacon menu bar icon may appear behind other icons or buttons. To move it to a visible spot, open Control Center, click Edit Controls, then drag the Beacon icon to your preferred position.
